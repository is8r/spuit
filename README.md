Spuit
==============

A Sass Mixin Library from is8r to is8r


## npm

install:

```
$ npm install spuit
```

import:

``` styles.scss
@import "node_modules/spuit/spuit";
```

## rails

---

For Development

## local server

```
$ yarn install
$ yarn run dev
```

## deploy

```
$ yarn run deploy
```
