Spuit
==============

A Sass Mixin Library from is8r to is8r

demo: https://spuit.12px.com/posts/components/

## npm

install:

```
$ npm install spuit
```

import:

``` styles.scss
@import "node_modules/spuit/spuit";
```

or

``` styles.scss
@import "~spuit";
```

## rails

import:

``` styles.scss
@import "spuit";
```

---

For Development

## local server

```
$ yarn install
$ yarn run dev
```

## deploy

npm:

```
$ npm publish
```

gem:

```
$ rake release
```
